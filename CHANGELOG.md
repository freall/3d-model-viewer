# 更新日志

本文档记录了3D模型查看器项目的所有重要变更。

## [1.0.0] - 2026-04-13

### 新增
- **初始发布**：完整的3D模型查看器网站
- **文件上传**：支持STL和3MF格式文件
- **3D渲染**：基于Three.js的实时渲染引擎
- **尺寸测量**：自动计算模型尺寸和体积
- **响应式设计**：桌面和移动端优化

### 功能
- STL文件解析器（二进制和ASCII格式）
- 3MF文件解析基础框架
- 模型旋转、缩放、平移控制
- 多光源照明系统
- 单位转换（毫米/厘米/英寸）
- 模型统计信息显示
- 暗黑/明亮主题切换

### 技术
- React 18 + TypeScript
- Three.js + @react-three/fiber
- Tailwind CSS + Framer Motion
- Vite构建工具
- GitHub Actions自动化部署

### 部署
- GitHub Pages自动部署
- 完整的CI/CD流程
- 生产就绪的构建配置

## 版本规范

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)。

### 版本号格式
- **主版本号**：不兼容的API修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 发布周期
- **正式版**：每季度发布一次主版本
- **特性版**：每月发布一次次版本
- **修复版**：根据需要随时发布

## 如何升级

### 用户
- 访问 https://freall.github.io/3d-model-viewer/
- 无需安装，自动使用最新版本

### 开发者
```bash
# 更新到最新版本
git pull origin main
npm install
npm run build
```

## 向后兼容性

### 保证向后兼容
- 所有API变更都会在次版本中进行
- 破坏性变更会在主版本中明确说明
- 废弃功能会提供迁移指南

### 不保证向后兼容
- 实验性功能
- 内部实现细节
- 未在文档中声明的功能
